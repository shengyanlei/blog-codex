# 02. 博客展示模块 (Blog Display)

## 1. 功能概述
核心阅读体验模块，包含文章列表展示、文章详情阅读、评论互动及侧边栏功能。

## 2. 页面需求

### 2.1 首页 / 列表页
- **Banner 区域**: 
  - 展示置顶或热门文章
  - 大图卡片设计
- **文章流 (Feed)**:
  - 无限滚动 (Infinite Scroll) 或 分页 (Pagination)
  - 文章卡片: 封面图, 标题, 摘要, 作者, 发布时间, 标签, 数据统计(阅读/点赞)
- **侧边栏 (Sidebar)**:
  - 个人简介 (Avatar, Bio, Social Links)
  - 热门文章列表
  - 标签云 / 分类列表
  - 站点公告

### 2.2 文章详情页 (/posts/:slug)
- **文章头部**:
  - 标题 (H1)
  - 元信息 (作者, 时间, 阅读时长, 字数)
  - 封面图 (Hero Image)
- **正文区域**:
  - Markdown 渲染 (支持 GFM)
  - 代码高亮 (PrismJS / Highlight.js)
  - 图片灯箱 (Lightbox) 点击放大
  - 目录 (TOC) 悬浮在左侧或右侧
- **文章底部**:
  - 标签列表
  - 版权声明
  - 上一篇 / 下一篇 导航
  - 点赞 / 收藏 按钮 (悬浮或固定)

### 2.3 评论区
- **列表展示**: 
  - 嵌套回复 (楼中楼)
  - 懒加载
- **发表评论**:
  - 登录用户: 直接输入框
  - 访客: 提示登录
  - 支持 Emoji, 简单 Markdown

## 3. 逻辑需求

- **SEO**: 
  - 动态设置 `<title>` 和 `<meta description>`
  - 结构化数据 (JSON-LD) 注入
- **性能优化**:
  - 图片懒加载 (Lazy Load)
  - 列表项虚拟滚动 (Virtual List) - 如果列表非常长
- **交互**:
  - 目录滚动监听 (Active Link 高亮)
  - 复制代码块功能
