# 04. 用户中心模块 (User Center)

## 1. 功能概述
管理用户个人资料、文章内容及系统设置。

## 2. 页面需求

### 2.1 仪表盘 (Dashboard)
- **数据概览**:
  - 总文章数
  - 总阅读量 / 点赞量
  - 近期数据趋势图 (Chart.js / Recharts)
- **快捷入口**:
  - 写文章
  - 审核评论

### 2.2 文章管理 (My Posts)
- **列表展示**:
  - 表格形式: 标题, 状态 (草稿/发布), 分类, 发布时间, 操作
- **筛选**:
  - 状态筛选, 关键词搜索
- **操作**:
  - 编辑, 删除 (软删除), 下线/发布

### 2.3 个人设置 (Profile Settings)
- **基本资料**:
  - 头像上传 (裁剪功能)
  - 昵称, 简介 (Bio)
  - 社交链接配置
- **账号安全**:
  - 修改密码
  - 绑定第三方账号
  - 登录设备管理

## 3. 逻辑需求

- **权限控制**:
  - 普通用户只能看到自己的文章
  - 管理员可以看到所有文章并进行管理
- **表单验证**:
  - 昵称长度限制
  - URL 格式校验
- **图片裁剪**:
  - 使用 `react-easy-crop` 或类似库，在前端完成头像裁剪后再上传
